introduction and notes
this is the list of current 
stable
 packages tracked by distrowatch, the latest versions and some notes and explanations where applicable. some developers choose to be very ambiguous about what is stable and what is still in development. package name capitalisation, even names themselves can vary between distributions. the capitalisation used here is equivalent to the "packagename" part in the "packagename-version.extension" that most open source developers use by convention, which sometimes differs from the "official" name (e.g. openssl-0.9.6f.tar.gz vs openssl). please 
report
 any errors or suggestions.
